### Accessing Badges

Use the global shortcut `/events` to open to the Events modal. The shortcut option will appear as you type `/events` into any conversation message.

![Global Shortcut](/events/events-shortcut.png)

Alternatively, users can access the modal by clicking the `Events` button in the Tightknit app home under the **Events** module section.

![Gamification Module](/gamification/gamification-module.png)

<Callout type="info">
  For non-admin users, the Events List modal displays all events published to
  any channel in the workspace.
</Callout>

### Allowing non-Admins to Create Events

By default, only Admins can create events.
If you would like to allow non-Admins to create events, you can do so by adding users to a [Group](/community/groups) which is assigned a [Permission Set](/community/permission-sets) with the Event Champion permissions enabled.
