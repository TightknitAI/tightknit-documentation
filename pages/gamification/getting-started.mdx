# Getting Started

<Callout emoji="✈️">
  Gamification is currently part of our pilot program. If you would like to
  participate, please contact [Support](mailto:support@tightknit.ai).
</Callout>

Ready to build a culture of reward and recognition in your community? With the Gamification module, you can:

- 🎖️ Create and give out customized Badges
- 🖼️ Generate framed award profile photos for your users
- 🎯 Set up milestones with achievements
- 🏆 Leaderboards (_coming soon_)

### Accessing Gamification

Admin users can access the module in the Tightknit app home under the **Gamification** module section.
