# Message Shortcuts

Tightknit comes with several [message shortcuts](https://slack.com/help/articles/360057554553-Use-shortcuts-to-take-actions-in-Slack) you can use to manage your community content from Slack.

## How to Access Shortcuts in Slack

<Steps>
### Step 1

<Tabs items={["Desktop", "Mobile"]}>
  <Tabs.Tab>
    **Desktop**: Hover over the message, and click the three dots icon `⋮`.
    <br />
    ![Slack message shortcuts
    hover](/slack-administration/shortcuts/message-shortcuts/message-shortcuts-hover.png)
  </Tabs.Tab>
  <Tabs.Tab>**Mobile**: Tap and hold the message</Tabs.Tab>
</Tabs>

### Step 2

The app shortcuts will be at the bottom of the menu. Find the options that have the Tightknit name and logo.

![Slack message shortcuts](/slack-administration/shortcuts/message-shortcuts/message-shortcuts.png)

If you don't see any Tightknit options, click `More message shortcuts...` and then search for "Tightknit" or the [name of the shortcut](#tightknit-shortcuts).

![Slack message shortcuts search](/slack-administration/shortcuts/message-shortcuts/message-shortcuts-search.png)

</Steps>

<Callout type="info">You cannot use shortcuts on ephemeral messages.</Callout>{" "}

## Tightknit Shortcuts

| **Name**                                   | **Description**                                                                                                                                                                                                                                         | **Where can I use it?**                                                                                                                         | **Who can use it?**                                                                                           |
| :----------------------------------------- | :------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ | :---------------------------------------------------------------------------------------------------------------------------------------------- | :------------------------------------------------------------------------------------------------------------ |
| Post to site                               | Syncs an entire message thread to the Tightknit companion site, including the post, any comments, file attachments, emojis, etc.<br/><br/>Has no effect on already synced posts.                                                                        | Slack message (Post)                                                                                                                            | Admin                                                                                                         |
| [Manage Content](#manage-content-shortcut) | Multi-purpose [menu](#manage-content-shortcut) for managing Tightknit content associated with the Slack message.                                                                                                                                        | Slack message                                                                                                                                   | _Dependent on the content_                                                                                    |
| Reply Anonymously                          | Sends an anonymous reply to a thread.                                                                                                                                                                                                                   | Any Slack message in a channel with [anonymous messages enabled](/member-engagement/anonymous-posting#enabling-channels-for-anonymous-messages) | All users                                                                                                     |
| Admin Experimental                         | Launches a menu of actions for an admin user to run a localized test. These actions are often a chance for admins to test out new Tightknit features in private, such as using AI to find related discussions. The results will only be visible to you. | Any Slack message in a channel with Tightknit                                                                                                   | Admin, the Event creator, or an Event host with the [permission](/community/permission-sets) to manage events |

### Manage Content Shortcut

The `Manage Shortcut` menu is a context-dependent menu that provides different actions based on the type of content associated with the Slack message.

| **Content Type**             | **Actions**                                                                                                                                                                                                                         | **Who can use it?**                 |
| :--------------------------- | :---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | :---------------------------------- |
| Post                         | • Sync thread to the site<br/>• Remove thread from site<br/>• Edit post title and slug <Callout type="warning">Modifying the slug will change the URL for the post and break any existing links or bookmarks to the page.</Callout> | Admin, or the author of the post    |
| Comment                      | • Sync thread to the site<br/>• Remove comment from site<br/>• Remove thread from site<br/>• Edit post title and slug<br/>                                                                                                          | Admin, or the author of the comment |
| Event (announcement message) | • Opens the [Edit Event](/events/manage-events) modal                                                                                                                                                                               | Admin, or owner of the event        |
