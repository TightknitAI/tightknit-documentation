# Engagement Settings

## Accessing Settings

To access the Agent settings for a channel, select the `Engagement Settings` option in the dropdown menu of the channel settings.

![Engagement Settings menu option](/channels-and-feeds/engagement/engagement-settings-option.png)

## Features

The following Engagement features are available for each channel:

- [Anonymous Posting](/member-engagement/anonymous-posting)
