# Branding Guide

| :--------------------- | :------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | :----------------------------------------------------------------------------------------------------------------------------------------- |
| Site Icon              | Appears in the top navigation bar alongside the site name, only if there is no **Site Horizontal Logo**. ## Branding Sets

A _branding set_ is a collection of branding token values that work with our design system. You can access your branding sets by clicking on the `Branding` button under the _Companion Site_ section of the admin settings. You can have up to 4 branding sets, with only one configured as the active branding set for your companion site.

<Callout type="info">The _Default Branding Set_ cannot be deleted.</Callout>
