# Navigation

Within the Navigation settings modal, you can configure the up to 3 custom links that will appear in the [navigation sidebar](/companion-site/pages/navbar-and-sidebar#sidebar) of the companion site.

Each navigation link must be input as label + URL, on a separate line in the following format:

```
Knowledge Base, https://example.com/
Contact Support, https://example.com/
Download the App, https://example.com/
```
