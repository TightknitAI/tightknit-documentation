# Directory Settings

You can access the settings for the [Directory page](/companion-site/pages/directory) on the companion site by clicking the `Directory` button under the **Profiles** module.

![Profiles Settings](/tightknit-site/profiles/profiles-module.png)

Within the Directory Settings, you can configure:

- **Directory page** - enable or disable the Directory page. When disabled, the route will be removed from all navigation menus and will be removed from SEO.
- **Max Display Limit** - the maximum number of members that will be displayed in the directory. By default, this is set to 10.

![Directory Modal](/tightknit-site/profiles/directory-modal.png)
