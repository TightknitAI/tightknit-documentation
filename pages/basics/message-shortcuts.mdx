# Message Shortcuts

import { Callout } from "nextra/components";

Use the Tightknit message shortcuts in Slack to manage your posts and comments in the community.

## How to Access Shortcuts in Slack

To access message shortcuts,

- Desktop: hover over the message, and click the three dots icon
- Mobile: tap and hold the message

![Slack message shortcuts hover](/message-shortcuts-hover.png)

The app shortcuts will be at the bottom of the menu. Find the options that have the Tightknit name and logo.

![Slack message shortcuts](/message-shortcuts.png)

If you don't see any Tightknit options, click "More message shortcuts..." and then search for _Tightknit_ or the [name](#tightknit-shortcuts) of the shortcut.

![Slack message shortcuts search](/message-shortcuts-search.png)

<Callout type="info">You cannot use shortcuts on ephemeral messages.</Callout>{" "}

| :----------------------- | :----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | :-------------------------------------------------------------------------------------------------------------------------- | :---------------------------------- |
| Post to community        | Syncs an entire message thread to the community, including the post, any comments, file attachments, emojis, etc.<br/><br/>Has no affect on already synced posts.                                                                                | Slack message (Post)                                                                                                        | Admin                               |
| Edit post title and slug | Updates the title and/or slug of the corresponding post in the community. <Callout type="warning">Modifying the slug will change the URL for the post and break any existing links or bookmarks to the page.</Callout>                           | Any message in Slack thread                                                                                                 | Admin                               |
| Delete from community    | Removes the post or comment from the community only. The original slack message is unaffected.<br/><br/>Deleting a post will also remove its comments. Deleting an individual comment will not affect the post or the remainder of the comments. | Delete a post:<br/>use on Slack message, or Tightknit bot reply.<br/><br/>Delete a comment:<br/>use on Slack message reply. | Admin, or the author of the message |
