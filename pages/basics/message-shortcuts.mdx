If you don't see any Tightknit options, click `More message shortcuts...` and then search for "Tightknit" or the [name of the shortcut](#tightknit-shortcuts).

## Tightknit Shortcuts

| **Name**                 | **Description**                                                                                                                                                                                                                                                         | **Where can I use it?**                                                                                                             | **Who can use it?**                 |
| :----------------------- | :---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | :---------------------------------------------------------------------------------------------------------------------------------- | :---------------------------------- |
| Post to site             | Syncs an entire message thread to the Tightknit companion site, including the post, any comments, file attachments, emojis, etc.<br/><br/>Has no effect on already synced posts.                                                                                        | Slack message (Post)                                                                                                                | Admin                               |
| Edit post title and slug | Updates the title and/or slug of the corresponding post in the Tightknit companion site. <Callout type="warning">Modifying the slug will change the URL for the post and break any existing links or bookmarks to the page.</Callout>                                   | Any message in Slack thread                                                                                                         | Admin                               |
| Remove from site         | Removes the post or comment from the Tightknit companion site only. The original Slack message is unaffected.<br/><br/>Deleting a post will also remove its comments, whereas deleting an individual comment will not affect the post or the remainder of the comments. | **Delete a post:**<br/>Use on Slack message, or Tightknit bot reply.<br/><br/>**Delete a comment:**<br/>Use on Slack message reply. | Admin, or the author of the message |
