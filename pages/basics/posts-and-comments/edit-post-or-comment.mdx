# Edit a Post or Comment

Edit your messages in Slack, and we'll update them in the community automatically!

## Slack Action

When you edit your Slack message body using the [Edit action](https://slack.com/help/articles/202395258-Edit-or-delete-messages#edit-a-message), your post or comment will
automatically be updated to reflect those changes in the community.

![Edit Slack message](/edit-message.png)

## Message Shortcut - Edit Post Title and Slug

You can also update the post title and slug using the Tightknit [message shortcut](../message-shortcuts.mdx).
