# How Items Are Synced

The following table outlines how and when Slack items are synced to your Tightknit site.

| Tightknit Site                                 | Sync Trigger Event(s) in Slack                                                                                                                                                                                                                                                                                                                                                          |
| :--------------------------------------------- | :-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Community logo                                 | Pulled from the Slack workspace logo only once when the community is created. If you need the logo updated, please reach out to the Tightknit team.                                                                                                                                                                                                                                     |
| Community member count                         | Pulled at intervals from the number of users in the Slack workspace                                                                                                                                                                                                                                                                                                                     |
| Feed description (_aka "About"_)               | • The channel’s Topic or Description is updated ([see how](https://slack.com/help/articles/201654083-Set-a-conversation-topic-or-channel-description)) <br/>• An admin clicks the "Sync" button in the feed settings menu in the Tightknit app home                                                                                                                                     |
| Feed topic                                     | • The channel’s Topic or Description is updated ([see how](https://slack.com/help/articles/201654083-Set-a-conversation-topic-or-channel-description)) <br/>• An admin clicks the "Sync" button in the feed settings menu in the Tightknit app home                                                                                                                                     |
| Posts <br/> Comments                           | • A user, or allowed Bot, sends a message or reply in a channel connected to a Tightknit feed ([see how](./posts-and-comments/create-post.mdx))<br/>• The author of a message, or an admin, uses the Tightknit message shortcut to create a Post in a channel that is connected to a Tightknit feed ([see how](./message-shortcuts.mdx))                                                |
| User profile (_name, picture, profile fields_) | A user's profile is created or updated when any of the following events occur:<br/>• The user authors a Slack message or reply that is synced to the Tightknit community<br/>• The user is @mentioned in a Slack message or reply that is synced to the Tightknit community<br/>• The user is @mentioned in the Topic or Description of a channel that is connected to a Tightknit feed |
